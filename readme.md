# Filament Tickets Plugin

A comprehensive ticket management system for FilamentPHP with real-time chat support.

## Features

- 🎫 Ticket Management
- 💬 Real-time Chat
- 📎 File Attachments
- 🔔 Email Notifications
- 👥 User Assignment
- 🏢 Department Management
- 🔒 Permission System
- 📱 Responsive Design

## Requirements

- PHP 8.1 or higher
- Laravel 10.0 or higher
- FilamentPHP 3.x
- Pusher (for real-time features)
